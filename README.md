# CYTADEL PROXY
## Установка и запуск
Клонируйте репозиторий
```bash
git clone https://github.com/Ggury/Cytadel.git
cd Cytadel
```
!Внимание. На данный момент бэкенд занимает порты 8000 и 3000. Перед запуском убедитесь что они свободны
```bash
sudo lsof -i :8000 -i :3000
```
### Запуск бэкенда
Убедитесь что у вас установлен docker
```bash
cd backend
docker compose build
docker compose up
```
### Запуск фронтенда
```bash
cd frontend
npm run dev
```

## Руководство пользователя
Перейдите на страницу регистрации
http://localhost:3000/register

Введите электронную почту, пароль и подтверждение пароля

После нажатия «Зарегистрироваться» бэкенд отправит письмо с ключом активации на указанную почту (на данный момент в console backend).
После этого можно зайти в личный кабинет используя свою почту и пароль и перейти в личный кабинет. 
Ключ можно получить либо в терминале с запущенным бэкендом.
### Запуск приложения 
Создайте среду
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```
